PROJECT — 01  /  FEATURED
Memory Retention Predictor
(Ebbinghaus Curve Tool)
A Python tool that estimates forgetting patterns using the Ebbinghaus Forgetting Curve — a mathematical model of how memory decays over time. Calculates retention probability at any interval and schedules optimal revision timing to maximise long-term recall. Applies real statistical theory to a practical learning optimization problem.
