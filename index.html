03 — PROJECTS

Things I've built.

PROJECT — 01  /  FEATURED
DataSense — Browser-Based CSV Analytics Tool
A fully offline, single-file HTML EDA tool that turns raw CSVs into clear intelligence. Drop any dataset and instantly get distributions, Pearson correlations (with corrected row-aligned pairedPearson()), IQR outlier detection, auto-binned histograms, scatter plots, and a natural-language rule-based insights engine — all running 100% client-side with PapaParse and Chart.js. Features k-NN and linear regression model training plus a downloadable HTML report.

PROJECT — 02
NexusSales — Revenue Intelligence Dashboard
End-to-end sales analytics pipeline: data cleaning → EDA → linear regression → reactive dashboard. Simulates a messy CRM dataset via a full Jupyter notebook (128 duplicates removed, 279 nulls imputed, 50 type errors fixed). Features animated KPI cards, category/rep filters, conversion rate analysis, and AOV trends.

PROJECT — 03
MNEMO — Smart Revision Scheduler
A memory retention predictor built on the Ebbinghaus Forgetting Curve with adaptive λ decay updates. Features an XP leveling system, streak tracking, focus history charts, and a "Needs Review" urgency engine — all deployed as a GitHub Pages PWA. Bridges real statistical theory with gamified habit-building.
